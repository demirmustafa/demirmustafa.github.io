Docker Installation on Raspberry Pi (Raspbian)

Docker Installation on Raspbian

p>Merhabalar.

Bu yazımda raspberry pi (raspbian) üzerine docker kurulumunu anlatacağım. SD card a raspbian işletim sisteminin olması gereken şekilde kurulduğunu varsayarak yazıma devam ediyorum.

Kurulum için öncelikle sistemden yönetici yetkisi almamız lazım.

sudo su

, ardından,

curl -sSL https://get.docker.com | sh

komutunu çalıştırıyoruz. Bu komut docker’ ı inirip kurulumunu gerçekleştirecektir.

İşlem tamamlandıktan sonra “docker -v” komutunu çalıştırarak docker kurulumunu teyit edebilirsiniz. Bu komut size kurulu olan docker versiyonunu gösterecektir. Ancak “dokcer images”, “docker ps”, gibi komutları çalıştırmaya çalışınca,

“Cannot Connect to the Docker Daemon” şeklinde bir hata almanız muhtemel. Bu hatayı ise aşağıdaki komutu çalıştırarak çözebilirsiniz.

sudo usermod -aG docker pi

Bu komut ile docker grup hesabına “pi” adlı kullanıcı eklenmiş olur. Raspbian da varsayılan olarak kullanıcı adı “pi” geldiği için sona “pi” yazdık. Eğer farklı bir kullanıcı eklemişseniz ya da farklı bir linux dağıtımı kullanıyorsanız (örneğin; ubuntu snappy core gibi) kullanıcı adını ona göre yazmanız gerekecektir.

 

Eğer docker servisinin sistem başlatılırken başlatılmasını istiyorsanız;

sudo systemctl enable docker

komutunu kullanabilirsiniz.

Sisteminizi ya da docker’ ı yeniden başlattığınız takdirde docker kullanıma hazır hale gelecektir.
